## 核心转化公式

**个人执行力公式：**
> 业绩 = 有效对话量 × 预约转化率 × 到院转化率 × 心态能量系数

**团队运营力公式：**
> 业绩 = 流量 × 转化 × 复盘迭代

两个公式互补：前者聚焦个人执行力，后者聚焦团队运营力。

## 4大核心方法论

1. **咨询约到8步法 + 多维约到法** —— 对话拆解、卡点分析的底层逻辑
2. **三段式安抚框架**（共情句 → 缓冲句 → 专业转入）—— 化解客户抗拒的标准结构
3. **培训落地三步法**（提炼3关键点 → 刻意练习3-7天 → 每日5分钟复盘）—— 让培训真正产生业绩
4. **AI话术生成三步法**（给AI设定角色 → 描述场景生成话术 → 筛选优化落地）—— 用AI源源不断生产话术

## 高效咨询4个优秀特质

1. **聚焦** — 聊天时全神贯注，一心不二用
2. **调频** — 能掌控自己的思维、情绪与行动
3. **执行** — 把"我觉得"变成"我来做"
4. **迭代** — 定期复盘，持续优化话术和流程

## 输出原则

1. **话输出即落地** — 所有话术可直接复制使用，不做空泛理论
2. **场景化** — 结合具体业务场景（口腔/医美/植发等）调整话术
3. **模块化组合** — 根据实际需求组合多个模块内容
4. **保持赖老师风格** — 接地气、有能量、直接实用

## 安装与使用

### 方式一：Claude Code（推荐）

**步骤1：** 把本仓库克隆到本地

```bash
git clone https://github.com/chenHongbin/medical-consult-conversion.git
```

**步骤2：** 将 `medical-consult-conversion` 文件夹放入 Claude Code 的 skills 目录

```bash
# 项目级（只对当前项目生效）
mkdir -p /你的项目路径/.claude/skills
cp -r medical-consult-conversion /你的项目路径/.claude/skills/

# 或用户级（对所有项目生效）
mkdir -p ~/.claude/skills
cp -r medical-consult-conversion ~/.claude/skills/
```

**步骤3：** 重启 Claude Code，或在对话中直接引用 skill 文件路径使用

**使用方法：**
- 在 Claude Code 中提问任何与医疗咨询转化相关的问题，例如：
  - "客户说再考虑考虑，怎么跟进？"
  - "给我3句安抚术前焦虑的话术"
  - "团队新人培训怎么设计？"
  - "怎么用AI生成回访话术？"

### 方式二：OpenClaw / 其他 AI Agent 工具

**通用安装步骤：**

1. 克隆本仓库到本地
2. 将 `medical-consult-conversion` 目录复制到你的 AI Agent 工具的 skills 目录下
   - OpenClaw 通常为 `~/.openclaw/skills/` 或项目内的 `.openclaw/skills/`
   - 具体路径请参考对应工具的官方文档
3. 重启 AI Agent 或刷新技能列表

**验证是否安装成功：**
向 AI 提问："帮我写一段应对客户价格犹豫的话术"，如果 AI 能引用本 skill 中的三段式安抚框架和具体话术，说明安装成功。

### 方式三：直接复制使用（最灵活，无需安装）

如果你不想配置 skill，可以直接复制本仓库中的任意 `.md` 文件内容，粘贴到 Claude、ChatGPT、Kimi、豆包等 AI 工具的对话框中作为上下文，然后提问。

**推荐 Prompt 模板：**
```
你是一位医疗咨询培训专家。以下是赖静茹老师的话术库资料：

[粘贴 scripts.md 中的相关内容]

请基于以上资料，帮我写一段应对[具体场景]的话术。
```

   ```

2. 将 `medical-consult-conversion` 文件夹复制到 OpenClaw 的 skills 目录
   - 通常为 `~/.openclaw/skills/` 或项目内的 `.openclaw/skills/`

3. 重启 OpenClaw，或刷新技能列表

### 使用方式

安装完成后，在对话框中像日常聊天一样提问即可，不需要加任何特殊命令。

**你输入：**
```
客户听完种植牙方案后说"太贵了，别人家才8000"，我怎么回？

**AI 回复（示例）：**
> 给你一段赖静茹老师风格的三段式回应：

> 然后再给你3套不同风格的话术（温柔共情型、专业权威型、朋友推荐型），你可以挑一个最顺口的用。

## 在 Work Buddy 中使用

Work Buddy 是 IMA（ima.qq.com）内置的 AI 助手。你可以把本仓库的资料导入 IMA 知识库，让 Work Buddy 基于赖老师的实战经验来回答你的问题。

### 使用步骤

1. 打开 IMA 客户端，进入你的知识库
2. 将 `medical-consult-conversion/` 目录下的所有 `.md` 文件批量导入知识库
3. 在 Work Buddy 的设置中，选择引用该知识库
4. 在对话框中直接提问

### 使用方式

**你输入：**
```
最近团队业绩下滑，新人培训完还是不会聊对话，怎么办？
```

**Work Buddy 回复（示例）：**
> 赖老师的建议是先从"培训落地三步法"入手：
> 
> 第一步：培训结束24小时内，让每个人提炼3个黄金关键点，并为每个点写3句参考话术，贴在电脑旁。
> 
> 第二步：接下来3-7天，每天刻意练习1个技巧，在实际对话中至少用3次，记录客户反应。
> 
> 第三步：每天下班前花5分钟复盘——今天用了什么技巧、用户反应如何、明天哪里可以优化。
> 
> 另外，你还可以用"团队管理四象限"分析一下，业绩差是意愿问题还是能力问题，针对性解决。

## 更多使用场景示例

### 场景1：自己状态很差，没动力跟进客户

**你问：**
```
最近业绩不好，很内耗，客户不回消息就自我怀疑，怎么调整状态？
```

**AI 会这样帮你：**
- 先给你一个"能量状态自检表"，5道题判断你是不是能量低谷
- 如果是低谷，给你具体的恢复动作：早晨先做3个老资源回访建立信心、午间必须休息、晚上写"进步清单"不写"失败清单"
- 再给你8大心法里的具体建议，比如"锚定精神主体性"——客户不回你不是你不行，是供需节奏错位

### 场景2：客户加了微信后不回消息

**你问：**
```
客户加微信后聊了几句就不回了，怎么激活？
```

**AI 会这样帮你：**
- 先告诉你客户沉默的真实心理：不是没兴趣，是怕被你缠、怕被套路、怕做了错误决定
- 然后给你分阶段激活话术：
  - 沉默1天左右：发案例资料，说"不急，你先看看"
  - 沉默2-3天：帮他做对比，说"不怕你比，希望你选之前看懂套路"
  - 沉默3天以上：主动退出推销姿态，送一份针对性的资料，说"哪怕不选我们，也知道怎么选"

### 场景3：用 AI 生成自己的专属话术

**你问：**
```
帮我生成一段应对"再考虑考虑"的话术，要口语化、像朋友聊天
```

**AI 会这样帮你：**
- 给你一个可以直接复制到 ChatGPT / Claude / Kimi 的 Prompt 模板
- 模板里已经写好了角色设定（15年经验的大健康咨询专家）和场景描述
- 生成后教你：3套话术中怎么选、怎么改成你自己的口头禅、怎么先在低意向客户上测试

### 场景4：朋友圈不知道发什么

**你问：**
```
给我一条今天可以发的朋友圈文案，要体现专业又不硬广
```

**AI 会这样帮你：**
- 给你一条现成的文案，比如科普干货型："最近好多客户问我同一个问题：XX项目到底痛不痛？今天统一回答3个最关心的..."
- 提醒你黄金发圈时间是晚上8点到10点
- 告诉你这条属于"专业人设"内容，占你朋友圈的30%即可，不要发太多
