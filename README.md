## 核心转化公式

**个人执行力公式：**
> 业绩 = 有效对话量 × 预约转化率 × 到院转化率 × 心态能量系数

**团队运营力公式：**
> 业绩 = 流量 × 转化 × 复盘迭代

两个公式互补：前者聚焦个人执行力，后者聚焦团队运营力。

## 4大核心方法论

1. **咨询约到8步法 + 多维约到法** —— 对话拆解、卡点分析的底层逻辑
2. **三段式安抚框架**（共情句 → 缓冲句 → 专业转入）—— 化解客户抗拒的标准结构
3. **培训落地三步法**（提炼3关键点 → 刻意练习3-7天 → 每日5分钟复盘）—— 让培训真正产生业绩
4. **AI话术生成三步法**（给AI设定角色 → 描述场景生成话术 → 筛选优化落地）—— 用AI源源不断生产话术

## 高效咨询4个优秀特质

1. **聚焦** — 聊天时全神贯注，一心不二用
2. **调频** — 能掌控自己的思维、情绪与行动
3. **执行** — 把"我觉得"变成"我来做"
4. **迭代** — 定期复盘，持续优化话术和流程

## 输出原则

1. **话输出即落地** — 所有话术可直接复制使用，不做空泛理论
2. **场景化** — 结合具体业务场景（口腔/医美/植发等）调整话术
3. **模块化组合** — 根据实际需求组合多个模块内容
4. **保持赖老师风格** — 接地气、有能量、直接实用

## 安装与使用

### 方式一：Claude Code（推荐）

**步骤1：** 把本仓库克隆到本地

```bash
git clone https://github.com/chenHongbin/medical-consult-conversion.git
```

**步骤2：** 将 `medical-consult-conversion` 文件夹放入 Claude Code 的 skills 目录

```bash
# 项目级（只对当前项目生效）
mkdir -p /你的项目路径/.claude/skills
cp -r medical-consult-conversion /你的项目路径/.claude/skills/

# 或用户级（对所有项目生效）
mkdir -p ~/.claude/skills
cp -r medical-consult-conversion ~/.claude/skills/
```

**步骤3：** 重启 Claude Code，或在对话中直接引用 skill 文件路径使用

**使用方法：**
- 在 Claude Code 中提问任何与医疗咨询转化相关的问题，例如：
  - "客户说再考虑考虑，怎么跟进？"
  - "给我3句安抚术前焦虑的话术"
  - "团队新人培训怎么设计？"
  - "怎么用AI生成回访话术？"

### 方式二：OpenClaw / 其他 AI Agent 工具

**通用安装步骤：**

1. 克隆本仓库到本地
2. 将 `medical-consult-conversion` 目录复制到你的 AI Agent 工具的 skills 目录下
   - OpenClaw 通常为 `~/.openclaw/skills/` 或项目内的 `.openclaw/skills/`
   - 具体路径请参考对应工具的官方文档
3. 重启 AI Agent 或刷新技能列表

**验证是否安装成功：**
向 AI 提问："帮我写一段应对客户价格犹豫的话术"，如果 AI 能引用本 skill 中的三段式安抚框架和具体话术，说明安装成功。

### 方式三：直接复制使用（最灵活，无需安装）

如果你不想配置 skill，可以直接复制本仓库中的任意 `.md` 文件内容，粘贴到 Claude、ChatGPT、Kimi、豆包等 AI 工具的对话框中作为上下文，然后提问。

**推荐 Prompt 模板：**
```
你是一位医疗咨询培训专家。以下是赖静茹老师的话术库资料：

[粘贴 scripts.md 中的相关内容]

请基于以上资料，帮我写一段应对[具体场景]的话术。
```

## 使用示例

### 示例1：客户说"太贵了"，想要话术

**你问 AI：**
```
客户听完种植牙方案后说"太贵了，别人家才8000"，怎么回应？
```

**AI 会基于 `scripts.md` 中的三段式安抚框架回复：**
> "王哥，您说这个数的时候，我就知道您不是差这点钱，是怕花冤枉了。谁的钱都不是大风刮来的，尤其是这种大几千上万的项目..."
>
> 然后给出3套不同风格的话术供你选择。

### 示例2：团队新人培训没效果

**你问 AI：**
```
新人培训完还是不会聊对话，怎么让培训真正落地？
```

**AI 会基于 `training.md` 回复：**
> 给你"培训落地三步法"：
> 1. 培训结束后24小时内，提炼3个黄金关键点 + 每人写3句参考话术
> 2. 接下来3-7天，每天刻意练习1个技巧，在实际对话中用3次
> 3. 每天下班前5分钟复盘：今天用了什么技巧、用户反应如何、哪里可以优化

### 示例3：自己状态很差，没动力跟进客户

**你问 AI：**
```
最近业绩不好，很内耗，客户不回消息就自我怀疑，怎么调整？
```

**AI 会基于 `mindset.md` 回复：**
> 先给你"能量状态自检表"，判断是不是能量低谷。如果是，启动恢复流程：
> - 早晨先做3个老资源回访，建立"赢"的感觉
> - 午间必须离开工位休息，不要边吃饭边工作
> - 晚上不写"失败清单"，写"今日进步清单"
> 再给你8大心法中的具体建议...

### 示例4：客户加了微信后不回消息

**你问 AI：**
```
客户加微信后聊了几句就不回了，怎么激活？
```

**AI 会基于 `followup.md` 回复：**
> 先分析客户沉默的6种心理（怕缠、怕不靠谱、怕被套路...），然后给你分阶段激活话术：
> - 沉默1天：发案例资料，不催
> - 沉默2-3天：帮他做对比，站在他立场说话
> - 沉默3天以上：主动退出推销姿态，送价值、给台阶

### 示例5：用 AI 生成专属话术

**你问 AI：**
```
帮我用AI生成一段应对"再考虑考虑"的话术，要口语化、像朋友聊天
```

**AI 会基于 `ai-empower.md` 回复：**
> 给你 AI Prompt 模板：
> ```
> 你是一位有15年经验的大健康行业咨询培训专家...
> 客户说"我再考虑一下"，请生成3套不同风格的回应话术...
> ```
> 并教你如何3选1、个人化改造、模拟演练、实战测试。

### 示例6：朋友圈不知道发什么

**你问 AI：**
```
给我一条今天可以发的朋友圈文案，要体现专业又不硬广
```

**AI 会基于 `wechat-moments.md` 回复：**
> 给你5大高转化模板之一（科普干货型）：
> "最近好多客户问我同一个问题：XX项目到底痛不痛？今天统一回答3个最关心的..."
> 并提醒你黄金发圈时间是20:00-22:00。
