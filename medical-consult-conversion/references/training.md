## 培训落地三步法

> 解决"培训时感觉都听懂了，一聊对话就不知道该怎么用"的核心痛点。

### 为什么学了却用不出来？

1. **信息过载，重点模糊** — 培训内容太多，分不清哪些是核心技巧
2. **缺乏场景化练习** — 知道理论，但不知道具体话术怎么说
3. **没有反馈机制** — 不知道自己用的对不对，效果如何

### 第一步：提炼3个黄金关键点（24小时内完成）

培训结束后立即做：
1. 回顾笔记，标出：最关键的3个技巧
2. 为每个技巧编写3句参考话术，或整理好技巧点匹配的对应内容
3. 把话术写在便利贴，贴在电脑旁，相关内容单独归档

### 第二步：刻意练习（持续3-7天）

上班前：
1. 选1个技巧重点练习
2. 在实际对话中刻意使用3次
3. 记录对话反应和转化情况

练习顺序：
- 先和同事模拟练习
- 再用于真实对话
- 从简单开始尝试

### 第三步：每日复盘（5分钟）

下班前花5分钟：
1. 今天用了哪个技巧
2. 用户反应如何
3. 哪里可以优化

**复盘模板：**
```
1. 今天使用[XX技巧]聊了[XX类型]对话
2. 对话情况[XX]，下次可以[XX]改进
3. 哪位同事运用[XX]很好，模拟借鉴
```

> 重复是技能之母。简单的事情重复做，坚持做，就会有效果。

### 常见问题解决方案

**问题1："话术背不熟怎么办？"**
- 制作"话术卡"随时查看
- 每天晨会互相演练

**问题2："对话/访客不按套路出牌怎么办？"**
准备3个万能应答话术：
- "这个问题问得好..."
- "很多xx也关心这个..."
- "我非常理解您..."

**问题3："效果不明显怎么办？"**
- 找主管或优秀同事分析+指导
- 录音分析自己的对话

## 团队管理四象限

> 带团队高转化，先识别成员状态，再针对性施策。

| 意愿 \ 能力 | 高能力 | 低能力 |
|---|---|---|
| **高意愿** | **授权型** — 给予肯定，团队扛把子，放手去干，授权充分 | **培养型** — 趁着有激情，多教实用干货，拆解对话，给话术技巧升级对话模式 |
| **低意愿** | **干预型** — 状态/能量/心力问题，提前一对一沟通，各种办法想一想 | **观察型** — 能拉起来就尽力拉，拉不起来则各自安好 |

### 各象限具体策略

**一、高意愿度高能力，必然转化高**
- 授权给予肯定，让他们放手去干
- 他们是团队标杆，让他们带动氛围

**二、高意愿低能力，培养多维能力**
- 有激情有心力有能量都好办
- 趁着这股劲儿，多教点实用干货，拆解对话，给话术技巧
- 他们绝对能帮你顶起来

**三、低意愿度高能力，重点干预**
- 这就是常说的"状态不好"——能力有，之前业绩也好，但就感觉不对
- 能量不足，心力不够，士气低迷
- 不管他们遇到什么问题，现在就是和你有关的问题
- 能起来，他好你也好团队好；起不来，团队或许不会废，但他可能"废了"
- 一定要提前干预，好好聊一聊

**四、低意愿度低能力**
- 有人就淘汰；实在没人，想办法拉
- 能拉起来也有拉起来的，拉不起来各自安好

> 管理的任务在于让人们的优势变得富有成效，让他们的弱点变得无关紧要。——彼得·德鲁克

### 用"左咨询 vs 右咨询"做团队诊断

详见 `mindset.md` § 左咨询 vs 右咨询。可把15组对照做成团队自查表，让每个人对照打分，找到改进方向。

## 导医客服培训法

### 核心方法：QA + 选择题轻松培训

传统培训方式密密麻麻的官方文档，新人看不完也记不住。改用**轻松娱乐化**的QA方式，让培训变得有趣好记。

### 文档模板：《XX医院明星导医18问》

**封面**：结合医院和企业文化风格设计，让企划部排版。
**格式**：QA + 选择题，通俗易懂，越看越喜欢。

### 内容框架

**基础版（1.0）— 必会基本功**

| 模块 | 问题示例 | 要点 |
|---|---|---|
| 仪容仪表 | "导医应该如何站？" "口袋里有五百万怎么站？" | 用假设场景让新人记住精神面貌 |
| 坐姿规范 | "在导医台应该如何坐？A.趴着 B.躺着 C.弯腰 D.优美地坐着" | 选择题方式，出其不意更容易记住 |
| 表情管理 | "有人进来该用什么表情？A.伤心 B.难过 C.大笑 D.微笑" | 00后新人接受度更高 |
| 接待话术 | "有病客进来该说什么？" | 标准话术 + 错误示范 |
| 挂号引导 | "挂号时应该怎么引导？" | 是否预约 + 推荐适合的主任 |
| 填病例引导 | "填病例应该怎么引导？" | 说明必要性，减少抗拒 |

**进阶版（2.0）— 难题处理**

| 场景 | 应对要点 |
|---|---|
| 收费处不缴费 | 安抚情绪 + 解释费用构成 + 请主管协助 |
| 担心治疗没效果 | 案例见证 + 医生资历 + 保障承诺 |
| 投诉服务态度 | 先道歉 + 了解具体情况 + 立即整改 + 回馈结果 |
| 等待时间过长 | 主动解释 + 提供茶水/杂志 + 告知预计时间 |

### 培训会议流程

**参与者**：导医组 + 经营主任 + 客服部主任（院长有空也参加）
**分组**：
- 一组负责"挑刺提问"
- 一组负责"回答应对"
- 互换角色，回答到所有人满意为止

**记录方式**：
- 投影仪现场记录（没有的话用电脑+录音）
- 每个环节3-5个问题
- 会后整理成文档 → 这就是医院专属培训教材

## 新人7天上岗计划

### Day 1：文化与制度

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | 医院介绍、企业文化、组织架构、规章制度 | 口头问答 |
| 下午 | 参观医院各部门、认识关键同事、熟悉工作环境 | 能说出各部门职能 |
| 作业 | 写"今日感受"+"三个疑问" | 上交主管 |

### Day 2：产品知识

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | 医院所有项目/产品介绍（分类、价格区间、适合人群） | 抽查5个项目能说出要点 |
| 下午 | 主任/医生介绍（擅长领域、资历、案例数量） | 能说出每位主任的亮点 |
| 作业 | 制作"主任能力对照表" | 上交主管 |

### Day 3：话术学习

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | 学习标准话术（开场、需求探询、价值塑造、邀约） | 背诵3套完整话术 |
| 下午 | 老带新旁听（看资深同事如何接待/打电话） | 记录10个学到的技巧 |
| 作业 | 和老同事模拟对话1次 | 主管旁听评估 |

### Day 4：系统操作

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | CRM系统使用、客户录入、跟进记录、预约登记 | 独立完成5条客户录入 |
| 下午 | 微信管理、朋友圈运营、快捷回复设置 | 设置好个人快捷回复 |
| 作业 | 整理自己的"快捷话术库" | 上交主管审核 |

### Day 5：实战旁听

| 时段 | 内容 | 考核 |
|---|---|---|
| 全天 | 跟岗实战（旁听资深同事接待客户/打电话/微信沟通） | 记录20个实战要点 |
| 重点 | 观察：开场怎么说、抗拒怎么化解、怎么邀约、怎么结束 | 写出"5个做得好的+5个可以改进的" |
| 作业 | 和老同事模拟完整接待流程2次 | 主管评估 |

### Day 6：模拟实战

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | 主管扮演客户，新人完整模拟接待流程 | 主管打分（话术/流程/态度） |
| 下午 | 独立接待/接听真实客户（主管旁听，必要时协助） | 完成3个真实客户接待 |
| 作业 | 写"第一次实战的感受和改进计划" | 上交主管 |

### Day 7：独立上岗+复盘

| 时段 | 内容 | 考核 |
|---|---|---|
| 上午 | 独立接待客户（主管远程关注） | 主管观察评估 |
| 下午 | 周复盘会议：本周学了什么、做得好的、要改进的 | 口头汇报 |
| 结束 | 主管一对一面谈：明确下阶段目标 | 签署目标责任书 |

### 新人带教黄金法则

1. **先完成再完美** — 第1周能上手就行，不需要精通
2. **每天一对一15分钟** — 主管每天抽15分钟专门辅导新人
3. **让新人第一天就出单** — 哪怕是小单，建立信心比赚钱重要
4. **老带新有奖励** — 带教的老同事要有动力
5. **第30天做能力评估** — 决定继续培养还是调整岗位

## 主管带教3步法

### 第一步：我说你听

> 主管讲，新人听。不要只是念文档，要：
> - 讲为什么（为什么要这么说/做）
> - 举例子（真实客户案例）
> - 说后果（说对了会怎样，说错了会怎样）

### 第二步：你做我看

> 新人做，主管看。不要打断，完整看完再点评：
> - 先肯定3个做得好的
> - 再指出2个可以改进的
> - 最后给1个具体建议

### 第三步：你做自己改

> 新人独立做，自己复盘：
> - 每天写"今日3个进步点"
> - 每周和主管一对一面谈
> - 第30天做独立能力评估

## 培训落地检查清单

### 培训前

- [ ] 明确培训目标（这次培训要解决什么问题）
- [ ] 准备培训材料（PPT/话术手册/案例集）
- [ ] 通知到位（时间/地点/需准备的物品）
- [ ] 场地布置（投影/音响/座位/茶水）

### 培训中

- [ ] 开场破冰（5分钟互动，调动气氛）
- [ ] 内容讲解（每20分钟休息一次，防止疲劳）
- [ ] 互动练习（至少3次实操演练）
- [ ] 现场答疑（收集问题，当场解答）
- [ ] 结束总结（3个要点回顾 + 作业布置）

### 培训后（关键！决定培训是否有效）

- [ ] 24小时内：发培训要点总结到工作群
- [ ] 3天内：新人提交"培训心得+行动计划"
- [ ] 7天内：主管一对一辅导，检验落地情况
- [ ] 30天内：数据检验（转化率/客户满意度是否提升）

> **培训不落地 = 白培训**
> 培训的终点不是"讲完"，而是"做到"。

## 内部知识库搭建

### 必备文档清单

| 文档 | 内容 | 更新频率 |
|---|---|---|
| **FAQ手册** | 客户常问的50个问题及标准答案 | 每月 |
| **话术宝典** | 各场景标准话术+优秀案例 | 每季度 |
| **案例集** | 成功转化案例（脱敏） | 每月 |
| **竞品分析** | 主要竞品的价格/优势/劣势 | 每季度 |
| **主任介绍** | 各主任擅长领域/案例/客户评价 | 有变动时 |
| **活动手册** | 所有活动的规则/话术/转化数据 | 每次活动后 |

### 知识库维护制度

1. **谁创造谁记录** — 每个咨询师都要贡献自己的成功案例
2. **每月评选最佳话术** — 奖励话术被收录最多的咨询师
3. **新人入职必读** — 知识库作为培训必修内容
4. **定期更新淘汰** — 过时的话术及时删除或更新
