## AI话术生成3步法

> 独家方法论，3步AI话术让业绩翻倍

### 第一步：给AI设定角色

**Prompt模板：**
```
你是一位有15年经验的大健康行业咨询培训专家。
你擅长：客户心理分析、高转化话术设计、抗拒化解。

我的业务场景：
- 医院类型：[医美/口腔/植发/不孕不育等]
- 客户画像：[年龄/性别/主要需求/常见抗拒点]
- 我的角色：[咨询师/网电咨询/客服主管]
- 当前问题：[具体场景描述]
```

### 第二步：描述场景让AI生成话术

**Prompt模板：**
```
请帮我生成3套不同风格的话术：

场景：[客户具体说什么/什么状态]
目标：[我想要达成什么结果]
风格要求：
- 第1套：温柔共情型（适合女性客户）
- 第2套：专业权威型（适合理性客户）
- 第3套：朋友推荐型（适合年轻客户）

要求：
- 自然口语化，不要太官方
- 每套话术包含：开场 + 核心内容 + 收尾
- 给出每套话术的适用场景说明
```

### 第三步：筛选优化落地

| 动作 | 说明 |
|---|---|
| 3选1 | 从AI生成的3套话术中选最适合自己风格的 |
| 个人化改造 | 加入自己的口头禅/医院的具体信息/真实案例 |
| 模拟演练 | 和同事模拟练习3遍以上 |
| 实战测试 | 先用在中低意向客户上测试效果 |
| 迭代优化 | 根据实战反馈调整，形成个人专属话术 |

### 用AI创造4大场景话术

**场景1：客户说"考虑一下"**
```
客户说"我再考虑一下"，我需要生成多套不同角度的回应话术。
我的项目：[具体项目]，客户类型：[XX]。
要求话术体现价值，不降价，不逼单。
```

**场景2：客户说"太贵了"**
```
客户说"太贵了"，我需要生成3套不同角度的回应话术。
我的项目：[具体项目]，市场价格区间：[X-Y]元，
我们的优势：[具体优势]。
要求话术体现价值而非降价。
```

**场景3：客户说"没时间"**
```
客户说"最近太忙了，没时间"，我需要生成激活话术。
要求帮客户重新评估"时间成本"，让对方意识到拖延的代价更大。
```

**场景4：客户说"你先告诉我用什么药/怎么做"**
```
客户在线上要求直接给方案/用药建议，我需要生成专业又安全的回应。
要求既体现专业度，又引导客户到院面诊，不直接给方案。
```

## 经典场景AI对照案例

> 以下展示同一个场景下，传统回应与AI赋能回应的差别。

### 场景：种植牙价格抗拒

**客户说：** "种植牙太贵了，能不能便宜点？"

**传统回应A：**
> "亲，我们这是德国进口材料，专家亲自操刀，这个价格已经很实惠了。"
> 
> ❌ 陷入价格辩论，强调贵的原因。

**传统回应B：**
> "现在有活动，可以给您打个95折。"
> 
> ❌ 直接让利，牺牲利润，客户还可能继续砍价。

**AI赋能后的回应：**
> "非常理解您对价格的关注。（共情）一个好的种植牙，本质上是一次对健康和生活质量的长期投资。（重塑框架：从'消费'到'投资'）"
> 
> "它让您未来20年都能畅快享受美食，不用担心咀嚼无力或尴尬的空缺，这种自信和舒适感是很难用价格衡量的。（描绘愿景，突出长期价值）"
> 
> "更重要的是，选择一位经验丰富的专家和可靠的材质，能极大降低后续维护的风险和成本，这其实是为您省下了未来很多不必要的开销。（转移焦点到'风险规避'）"
> 
> "这样吧，我请专家为您做一个详细的种植方案，让您清清楚楚看到每一分钱的价值所在。您看周二还是周三方便过来呢？（给出选择，引导到店）"

**你看出差别了吗？**
AI不是教你敷衍，而是教你如何把客户的思维从"价格"引导到"价值"，从"成本"引导到"收益"。这种对话深度和专业性，直接拉开了你与普通咨询师的差距。

## AI头脑风暴会议流程

### 适用场景

- 业绩连续下滑，团队没思路
- 新活动/新项目启动，需要创意
- 竞品出了新策略，需要应对
- 月度/季度复盘，需要改进方案

### 会议流程（60分钟）

**Step 1：问题输入（10分钟）**
> 每个人说出当前最头疼的3个问题（不讨论解决方案，只描述问题）
> 主管记录所有问题，分类汇总

**Step 2：AI brainstorm（20分钟）**
> 把问题输入AI，让AI生成：
> - 每个问题的3个解决方向
> - 每个方向的优劣势分析
> - 推荐的优先级排序

**Step 3：团队讨论（20分钟）**
> 基于AI的建议，团队讨论：
> - 哪个方向最可行？
> - 我们有哪些资源可以用？
> - 谁负责什么？
> - 什么时候验收？

**Step 4：行动计划（10分钟）**
> 输出：
> - 3个必做行动项
> - 负责人 + 截止日期
> - 验收标准

### AI Brainstorm Prompt

```
我们是一个[医美/口腔等]咨询团队，目前遇到以下问题：

问题列表：
1. [问题1]
2. [问题2]
3. [问题3]

请帮我：
1. 分析每个问题的根本原因
2. 针对每个问题给出3个具体可行的解决方案
3. 评估每个方案的投入产出比
4. 给出执行的优先级排序
5. 每个方案需要谁参与、大概多久见效
```

## 每日AI工作流

> 从小处着手，把AI深度融入日常工作流。

### 上班前
- 让AI帮你生成当日的正能量口号，鼓舞士气
- 快速浏览AI生成的今日行业热点/科普素材

### 遇到难题时
- 复制粘贴客户对话，寻求AI的话术支持
- 让AI分析客户性格和抗拒点，给出精准出击建议

### 下班前
- 用AI快速复盘今日聊天，总结得失
- 让AI总结：今天在哪类问题上浪费了最多时间？哪些机会点没把握好？

### 睡前
- 让AI根据你行业的最新动态，帮你生成明日可发的专业观点朋友圈
- 让AI基于明日客户预约情况，生成个性化跟进话术

### 3步上手AI，明天就开始提效

**第一步：改变心态，从"排斥者"变成"探索者"**
- 不要想着"AI会不会取代我"，而是想着"我怎么利用AI取代那些不会用AI的人"
- 主动去了解元宝、DS、豆包、Kimi等主流工具，从免费版开始玩起

**第二步：从小处着手，解决具体问题**
- 今天就用AI帮你优化一句打招呼的话术
- 明天让它帮你分析一个刁难问题的回复
- 后天让它帮你生成一个朋友圈文案
- 从解决一个个小问题开始，建立信心

**第三步：刻意练习，形成你的"AI工作流"**
- 把AI深度融入日常，形成固定节奏
- 很快你会发现，AI就像是你身边的一位超级助理、策略大师和心理咨询师，三者合一

> 未来已来，只是分布得还不均匀。——凯文·凯利

| 工具 | 用途 | 使用场景 |

| 工具 | 用途 | 使用场景 |

5. **客户问题太刁钻** → 把客户原话+你的回复输入AI，让它帮你找漏洞和优化点

## AI发圈与内容营销

### AI朋友圈内容公式

> **有价值的内容 + 个人化表达 + 轻微引导 = 高互动朋友圈**

### 5类高转化朋友圈内容（AI生成）

**类型1：客户见证**
```
Prompt: "写一个客户见证朋友圈，客户做了[项目]，
效果很满意，主动发来感谢。语气像朋友分享，
不要太广告。包含：客户原话（改编）+ 效果描述 + 轻微引导"
```

**类型2：科普干货**
```
Prompt: "写一个关于[项目]的科普朋友圈，
3个大多数人不知道的真相，用轻松有趣的语气，
让人看了觉得'原来如此'，树立专业形象"
```

**类型3：日常+业务融合**
```
Prompt: "我今天[做了什么事，比如喝咖啡/健身/陪孩子]，
把这件事和[工作感悟]结合起来，
写一条有生活气息又有专业深度的朋友圈"
```

**类型4：限时福利**
```
Prompt: "写一条活动预告朋友圈，[活动名称]，
限时X天，前XX名有额外福利。
制造稀缺感和紧迫感，但不要硬推销"
```

**类型5：团队/个人IP**
```
Prompt: "写一条展示我们团队专业度的朋友圈，
[具体事件，比如培训/获奖/客户好评]，
体现团队实力和个人成长"
```

### AI视频内容创作流程

1. **豆包写脚本** — 输入主题和风格要求，生成视频脚本
2. **即梦生成视频** — 用脚本中的提示词在即梦中生成视频素材
3. **剪映剪辑** — 加字幕、音乐、转场
4. **发布到视频号/抖音** — 引导客户加微信

## AI数据分析与复盘

### 每日数据AI复盘Prompt

```
这是我今天的工作数据：
- 总对话量：XX
- 有效对话：XX
- 有效对话率：XX%
- 预约量：XX
- 预约转化率：XX%
- 实际到店：XX
- 到店率：XX%

请帮我分析：
1. 哪些数据高于/低于正常水平？
2. 可能的原因是什么？
3. 明天应该重点改进哪个环节？
4. 给我3个具体的改进建议
```

### 周度复盘AI Prompt

```
这是我们团队本周的数据：
[粘贴团队每个人的关键数据]

请帮我：
1. 找出本周Top 3最佳实践（谁做了什么效果好）
2. 找出本周Top 3改进机会
3. 生成下周团队重点行动计划
4. 给主管的3条管理建议
```

### 关键提醒

> **AI是辅助工具，不是替代品**
> - AI生成的话术要经过个人化改造才能用
> - AI分析的数据要结合实际情况判断
> - 最终的客户沟通还是要靠人的温度和判断力
> 
> **最好的咨询师 = AI的效率 + 人的温度**
