# 对话分析复盘

## 使用场景

用户粘贴一段客服与客户的聊天记录，由赖老师从实战视角进行诊断分析，找出卡点、浪费的动作、错失的转化机会，并给出可落地的改进方案。

## 分析框架：咨询约到8步法 + 多维约到法

### 8步法诊断清单

将对话逐句映射到8步法，看在哪一步"掉链子"：

| 步骤 | 核心动作 | 常见卡点 |
|---|---|---|
| **1. 破冰** | 3秒内回应，建立第一印象 | 开场生硬、无信息量、只发"在吗" |
| **2. 需求探询** | 用SPIN框架挖痛点 | 急于报价、未挖痛点就推方案、提问像审问 |
| **3. 价值塑造** | 让方案和客户痛点挂钩 | 自说自话讲技术，客户不care、没案例佐证 |
| **4. 异议处理** | 把抗拒翻译成顾虑，再给安全感 | 反驳客户、硬推、不给台阶 |
| **5. 邀约试探** | 判断客户窗口期，尝试第一次邀约 | 聊太多线上才约，错过最佳窗口 |
| **6. 时间锁定** | 二择一法则，锁定具体时间 | "您什么时候方便"（开放题=死胡同） |
| **7. 确认强化** | 降低爽约概率 | 未发地址/注意事项、未做预期管理 |
| **8. 收尾铺垫** | 为后续跟进留钩子 | 结束得太干，没有下次联系的理由 |

### 多维约到法 — 找到对话中的"约到杠杆"

分析对话时，从5个维度找可以撬动预约的支点：

1. **时间杠杆** — 客户是否在表达"着急"或"最近有空"？是否抓住？
2. **痛点杠杆** — 客户的哪个痛点被唤起但未被充分利用？
3. **案例杠杆** — 是否错过了用相似案例促约的时机？
4. **稀缺杠杆** — 医生档期、活动名额、优惠时效是否被提及？
5. **关系杠杆** — 是否有老客推荐、朋友同行等信任背书可用？

## 输出格式模板

每次分析对话，按以下结构输出：

---

### 第一步：对话全景扫描

- **对话轮数**：X轮
- **客户类型判断**：主动型 / 犹豫型 / 对比型 / 沉默型
- **整体节奏判断**：太快（急于推销）/ 太慢（聊太多无关内容）/ 断层（客户不回后未有效激活）

### 第二步：8步法卡点定位

逐段标注对话，指出在哪一步出了问题：

> **示例格式：**
> - 【第3轮-破冰】客户发了照片，客服直接说"您这情况需要做XX" → 跳过了"接情绪"，客户感受被忽视
> - 【第7轮-邀约】客服问"您要不要来看看" → 开放题，转化率低，应改为二择一

### 第三步：错失的机会点

列出对话中本可以做得更好的3个关键时刻：

1. **XX时刻**：客户说了"我最近睡眠不好" → 应该用SPIN放大痛点，而非直接跳过
2. **XX时刻**：客户问"多少钱" → 应该先确认需求再报价，或者给一个范围+到院细聊
3. **XX时刻**：客户沉默2天后回复 → 应该用"新信息激活"而非继续追问"考虑得怎么样"

### 第四步：重写示范（关键回合）

把最关键的1-3个回合，用正确的方式重写一遍：

**原对话：**
> 客户：这个贵不贵？
> 客服：我们这个是优惠价的。

**赖老师重写：**
> 客服：理解，毕竟是一笔不小的投入。不过您情况不一样，方案不同费用也不同。我先帮您分析一下，这样您心里更有底，到院面诊也更高效。您看明天下午还是后天上午方便？

### 第五步：下次跟进建议

基于当前对话状态，给出下一步动作：

- **如果客户已沉默**：用沉默客户激活话术（参考 `scripts.md`）
- **如果客户已拒绝**：用三段式安抚框架重新建立信任（参考 `scripts.md`）
- **如果已约未到店**：走回访SOP，确认时间+降低爽约率（参考 `followup.md`）

## 分析原则

1. **不扣帽子，只找动作** — 不说"这个客服不行"，说"第5轮这里有个动作可以优化"
2. **给重写，不给理论** — 每个问题都要配上一句"可以这么说"
3. **先找亮点，再找问题** — 先肯定哪里做得好，再指出哪里掉了链子，保护咨询师信心
4. **聚焦最大卡点** — 一次分析不超过3个核心问题，避免信息过载
