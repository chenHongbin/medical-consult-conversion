## 内耗识别与破解

### 内耗3大表现

| 表现 | 症状 | 破解方法 |
|---|---|---|
| **过度反思** | 客户没回复就怀疑自己；被拒3次就觉得自己不行；开始否定职业选择 | 接受"销售是概率游戏"；80%成交需要5次跟进；关注过程指标而非单次结果 |
| **情绪疲劳** | 害怕聊不到有效；看到未读心跳加速；晚上失眠回想白天对话 | 早晨"赢一次"建立信心；午间充电（运动/冥想/短视频）；晚上复盘进步而非纠结失败 |
| **无效努力** | 拼命加人但转化率极低；把时间花在"容易但无效"的事上；用战术勤奋掩盖战略懒惰 | 优化流量渠道吸引精准客户；优化销售流程减少无效沟通；学习新话术而非反复改旧话术 |

### 停止内耗3步法

> **第一步：接受概率**
> - 你不可能让每个客户都买单
> - 关注"有效对话量 x 转化率"，而非单客成败
> - 设定过程目标（每天10个意向跟进），而非只盯结果

> **第二步：管理能量**
> - 早晨先做容易赢的事（老资源回访/预约确认）
> - 午间必须休息，持续工作=效率递减
> - 晚上写"今日进步清单"而非"失败清单"

> **第三步：优化策略**
> - 爱因斯坦："疯狂就是重复做同一件事，却期待不同结果"
> - 流量问题 → 换渠道/优化引流内容
> - 转化问题 → 学新话术/优化流程
> - 心态问题 → 减少自我攻击/提升状态

### 能量状态自检表

每天上班前问自己：

- [ ] 昨晚睡够7小时了吗？
- [ ] 今天有没有一件让我期待的事？
- [ ] 我愿意主动给客户打电话吗？
- [ ] 我相信今天会有成交吗？
- [ ] 遇到困难我会想办法解决而不是抱怨吗？

**3个以上否 = 能量低谷，需要启动能量恢复流程**

## 网咨必看8大心法

### 01 锚定精神主体性，筑牢情绪"不受力"防护盾

患者沉默、沟通卡壳，本质是供需节奏错位，绝非你"服务失职"。

真正的专业，是守住精神主体性：你负责用专业解需求、用真诚传价值，患者负责消化信息、按自己的意愿做选择，各归其位就好。

别被"已读不回"带偏情绪，别把成交压力当自我评判标准。这份"不受力"的从容，会让你更显专业可信。

### 02 接纳卡点穿行，在主体性中沉淀业绩底气

能以个人意志直面沟通不顺，才是涨业绩、快成长的关键。

"已读不回"后别只顾失落，跟着赖老师复盘：
- 是不是没摸清患者对疗效、恢复周期的顾虑？
- 是不是没讲清咱们专家优势和与别家的差别？
- 是不是没共情？疗效强调到位了吗？
- 我有传递出"我就是你唯一希望"的信念感了吗？
- 我是不是只在讲疾病，一顿输出，忘记"有图有真相"了？

跳出情绪俯瞰问题，提炼方法，沟通掌控力会越来越强。

### 03 "心硬"守边界，共情有温度

面对患者的顾虑、质疑，急于反驳或自我否定都是内耗。

"心硬"是用意志守住边界，不被患者负面态度消耗；"共情"是咱们的底色。

- 患者怕没效果 → 拿同症案例、医生评估拆解，不催预约
- 患者反复要优惠 → 说清定价和保障，疗效优势与差异
- 患者失约或没回复 → 定期发送健康科普、客情正能量与效果案例，不频繁打扰

### 04 拒绝"厌蠢"内耗，聚焦可掌控的主体性动作

患者认知局限不是你的责任。分享一个"意志过滤器"思维：

你讲清专业知识、疗效优势、口碑案例等价值再有积极引导就够了，剩下的交给对方选择。把烦躁的精力用来磨话术、攒案例、优化回复。

### 05 耐住沟通周期，以意志沉淀核心转化力

健康决策本就偏"慢"，患者犹豫、拖延很正常。急于求成，一次沟通无果就泄气，反而错失后续转化机会。

精准拆解决策痛点：
- 预算不够 → 提灵活付款
- 没时间 → 协调周末出诊
- 对比竞品 → 说清专属随访优势
- 怕被骗 → 给案例、给保障

### 06 放下完美执念，接纳"七分足够"的主体性松弛

硬要"让所有人满意"，把他人评价当唯一标尺，丢了工作主体性，越累业绩越差。

网电咨询工作的本质是帮有需要的人解决问题。你能让多数患者感受到专业、化解健康困惑、感受到你们医院的口碑优势疗效、预约到院，就值得肯定。

别纠结1%的负面反馈，接纳不完美，守住"七分就好"的松弛感。

### 07 以硬目标锚定方向，用行动消解情绪内耗

提心力、涨业绩，从定清晰的阶段目标开始。比如本月定这几个目标：

- 打磨3套高频话术（应对效果质疑、费用顾虑、竞品对比）
- 积累5个同病种高转化案例（标注突破口）
- 提升未预约患者二次跟进回复率，哪怕10%
- 吃透2类最影响你预约的病种或者预约卡点
- 不太敢打电话或者电话量不够，在这方面精准加强

聚焦这些具体动作——深耕专业知识，复盘沟通得失，情绪内耗会自然消散。

### 08 不向自我"开枪"，接纳不完美的工作状态

偶尔没分析对疾病、报错价格、说错疗程、漏传预约须知，或是应对患者情绪爆发不够从容，都是正常成长插曲。

别因一次爽约就自我否定，别因一句"你没讲清"就怀疑专业。你的价值，从不由单一结果定义。

网电咨询工作核心是"真诚+专业"。每天都在进步并且时刻传递健康价值、坚守初心，就值得肯定。放过自己，才能保持长久热情与心力。

## 高效咨询4个优秀特质

### 一、学会发问：把"死对话"变成"活互动"

高手客服永远不会只回答问题，而是"借题发挥"，用提问推动互动。

**三步法：先共情 → 再讲价值 → 再引导发问**

**1. 共情回应**
> "我理解您的顾虑，价格确实是大家最关心的问题之一。"
> "能理解，现在大家做项目都比较谨慎，都想先了解清楚再决定。"

**2. 讲优势价值**
> "我们这边的项目，价格是根据检查项目和方案来定的，主要是帮您先查清楚问题根源，再做针对性方案，这样比盲目做项目更省钱、也更安全。"

**3. 引导性发问**
> "方便问一下，您主要是想解决哪方面的问题呢？是之前有检查过，还是这次是第一次了解？"
> "您现在最在意的是价格、时间，还是效果这块呢？我好根据您的情况给您一个最合适的建议。"

> **顾客问"价"，你先给"值"；顾客要"答案"，你多给"问题"。**

### 二、学会思考：同样是话术，有人被挂电话，有人被加微信

**1. 渠道不同，开口话术也要不同**

- **电商渠道**（顾客主动来问，意向强）：
  > "您好，我看您是咨询【XX项目】的，方便跟我说说，您主要是想改善哪方面的问题吗？"

- **新媒体渠道**（顾客多是"好奇来看看"，需先建信任）：
  > "您好，我是XX，您可以叫我xx，方便跟我说说，您现在主要是担心哪方面的问题呢？我这边可以先帮您做个简单的情况了解和分析。"

**2. 自报家门被挂电话？先调整"顺序"**

正确顺序：**先给价值 → 再报家门 → 再约下一步**

> "您好，之前联系过/你是要解决A问题还是B问题？刚刚在后台看到您有咨询过【xx问题】，很多像您这样的情况，通过前期的专业评估，后面在治疗和花费上都能少走很多弯路。我这边可以先简单了解一下您的情况，看您是适合线上指导，还是加微信发图给您详细介绍，您是a情况还是b情况？"

**3. 每天做一个简单复盘**

今天有3个没约到的顾客，各自卡在了哪里？是价格问题、信任问题、时间问题，还是话术顺序问题？

> 不会思考的客服，只会抱怨"顾客难搞"；会思考的客服，会问自己："这次没约到，是想教会我什么？"

### 三、刻意练习：把优点放大，把短板补齐

**1. 先发现自己的优点，再刻意放大**

问自己：我是不是比较有耐心？声音比较温柔？逻辑比较清晰？比较会安慰人？

找到1-2个优点，在每一条对话中刻意放大。

**2. 接纳不足，"有针对性地练"**

- 语速太快 → 刻意放慢，每句话多停顿一秒
- 容易被顾客带节奏 → 事先写好3-5个"拉回节奏"的过渡句
- 不太会处理价格异议 → 多练几套"价格+价值"的标准话术

练习方法：自己录音回放找问题、和同事互相模拟点评、把优秀同事聊天记录整理成"话术库"。

> 优秀不是天生的，是"刻意练习"的结果。你每天多练10分钟，一个月就是300分钟，三个月就会有质的变化。

### 四、遇到挑战，迎接挑战：把"难搞对话"当成训练场

**1. 每一次"没约到"，都是一次成长机会**

问自己：
- 顾客真正的顾虑是什么？我有没有挖出来？
- 哪一句话可能让顾客有压力了？
- 如果重来一次，我会换一种说法吗？

**2. 做业绩没有不累的，但可以更高效**

- 多听：优秀同事的录音、顾客的真实反馈
- 多学：话术、专业知识、沟通技巧
- 多练：每天刻意练几段话术，练到脱口而出
- 多总结：每天5分钟复盘，每周一次小总结

> 人们眼中的天才之所以卓越非凡，并非天资超人一等，而是付出了持续不断的努力。——格拉德威尔《异类》

## 左咨询 vs 右咨询

> **你是左咨询还是右咨询？** 以下15组对照，可用于自我诊断或团队激励。

| 场景 | 左咨询 | 右咨询 |
|---|---|---|
| **八点上班** | 踩点打卡，到岗吃东西、照镜子、刷朋友圈，才想进入状态 | 提前到岗，整理好一切，定好今日目标，有条不紊按计划进行 |
| **三天没成交** | 对话没质量，对方没意向，也急也烦，但只说无行动 | 自我总结分析，找问题，找主管，想办法，用行动解决问题 |
| **主管谈话** | 以为主管对自己有意见，只有解释，没有行动 | 感谢主管指引，寻求帮助，努力改进 |
| **状态不佳** | 带情绪聊对话且影响同事，总寻求安慰，仿佛全世界跟自己作对 | 迅速想办法自我调整，梳理情绪，避免影响同事和业绩 |
| **业绩下滑** | 反正不垫底，无所谓 | 想办法和第一名缩短距离，不服输，一定想办法提升 |
| **有人加班** | "成天加班，害我不加班都被领导说，那么积极表现给谁看" | "大家都这么积极，我也不能落下。多付出，相信一定会有收获" |
| **通知开会** | 最烦开会，下班还有事，想请假 | 做足准备，工作总结，数据分析，准备好问题寻求探讨和解答 |
| **会议中** | 坐后面，看手机，刷朋友圈，左顾右盼，心不在焉 | 坐前面，精美笔记本，用心听，认真记，积极回应，主动交流 |
| **对接群沟通** | 无礼貌用语，不积极回应，不顾他人感受 | 常用：您好、请、帮忙、麻烦、谢谢、辛苦了，常常换位思考 |
| **集体活动** | "成天占用下班时间，搞这些，烦啊" | 积极参与，感恩公司，拍照发朋友圈 |
| **有人离职** | 疑心重重：好好的为什么忽然走了，是被开除还是被挖了 | 有点可惜，但很正常，天下没有不散的宴席，我只管做好自己 |
| **有新人报道** | 给脸色看，不闻不问，问了也懒得搭理 | 主动大方问候，表示欢迎，让其感受团队的温暖 |
| **外部挖角** | "哇，给保底呢，比这里高好多，赶紧行动" | "公司对我很好，主管对我也很重视，不能辜负主管的信任" |
| **晚班回宿舍** | 走路大声，洗刷吵吵闹闹，不考虑别人感受 | 考虑他人感受，轻手轻脚，想着早班同事已经睡了不要吵到他们 |
| **看到本文** | "说的什么啊，不会是在说我吧，好像有些地方跟我有点像" | "优秀的人都那么努力，那我也必须加快脚步。珍惜时间，努力进取" |

> **努力很累，但不努力造成力不从心的自己和现状更累！**

## 43条职业忠告

> 精选自赖静茹老师《咨询师的自我修养》，覆盖工作、人际、生活三大维度。

### 工作态度（1-12）

1. 要相信做咨询有前途，不然这么聪慧的你怎么会选择做咨询
2. 既来之则安之，时间不等人，别边做边后悔
3. 新手脱颖而出最快办法：找到业绩第一的人，全部模仿她
4. 新人脱颖而出最快办法：业绩，业绩，业绩
5. 业绩暂时无法出彩，工作态度一定要好——积极、向上、主动
6. 不要以为自己是无名小卒，总有人会在注意着你
7. 领导在和领导不在，一个样。业绩是自己的
8. 领导喜欢你，用成绩让他知道他没看走眼
9. 脱颖而出很简单：工作态度好、责任心强、业绩好、忠诚度高、感恩的心
10. 不要随便跳槽：学不到东西、薪资不会高、名声不会好
11. 要离职就光明正大，不要用家人开玩笑当借口
12. 离职了不要说前公司坏话，谁都不敢用你

### 业绩提升（13-30）

13. 收集成功案例好好学习，收集失败案例好好分析，不出五天业绩一定上去
14. 业绩不行，想想是能力不行还是内心在抗拒——方向方法很重要
15. 告诉自己：我是要干大事的人，别被鸡毛蒜皮牵绊
16. 想辞职是正常的，但别真的辞——在这家公司遇到的问题，别的公司也会有
17. 辞职了还要应聘同个岗位，别辞了——要辞就去挑战更高岗位
18. 脑子很重要，不要做煽风点火的人，也不要那么容易被煽风点火
19. 业绩好就勇敢表扬自己，自信就是这么来的
20. 业绩不好，把责任推给别人也提升不了业绩，不如多想想自己能让结果更好
21. 办法总是有的，就看你愿不愿意——同行交流、读书、百度、请教
22. 老板不会因为关系好就多给你发1000元，但可能因为你业绩突破多奖励1000元
23. 看看关系最好的同事业绩如何——要跟比自己强的人交朋友，遇强则强
24. 领导批评你 = 你被重视 + 有机会提升 + 错误的代价不用你买单
25. 下班不急着走，多总结今天的对话，日积月累收获很大
26. 懒得打回访时，你的竞争对手正在给你的客户打电话
27. 打了三次回访不接就放弃 = 前面三次白打了，出气的最好办法是打到他来为止
28. 讨厌一个人，最好的办法就是业绩超过他
29. 看不惯主管，就去跟领导说你来当，没这本事就闭嘴好好做业绩
30. 你在这讨厌的任何人和事，去别的地方同样会出现——真相是：你太穷了，口袋穷心也穷

### 人际与生活（31-43）

31. 聊对话不开心的时候，休假去干喜欢的事，如果喜欢的太贵就赶紧回来赚钱
32. 不要为老板瞎操心，你操心的是你的业绩好不好
33. 在每一个群里，讲话都注意态度和礼貌——你是代表你们部门
34. 总感觉被埋没，不是被埋没而是不出彩——想成为什么先去做
35. 聊对话一定要集中注意力、聚焦，拿下的机会很大，松懈就没了
36. 定期优化自己的快捷方式、思路、话术——舍得给大脑投资
37. 想不出好的话术，直接复制别人的或在别人基础上改编升级
38. 主动找主管甚至大领导聊天，有野心是好事
39. 一时之气闹不开心，事后发现不应该就主动道歉——退一步海阔天空
40. 哪里是状态不好影响业绩，不过是追求不够大、退路还很多
41. 遇到领导不要假装看不见，大声打招呼
42. 有点业余爱好，写作、阅读、健身、瑜伽——生活多彩

## 团队激励话术

### 早会激励（5分钟）

> "昨天的成绩不管好坏都翻篇了。今天是个全新的开始，我们只做一件事：比昨天的自己好一点。过程做好，结果自然来。大家有没有信心？"

### 个人低谷激励

> "业绩波动是正常的，姚明也有投不进球的时候。但我看到的是：你态度一直在线，这很重要。来，我们一起分析一下昨天的对话，看看是哪个环节可以优化。"

### 团队竞赛激励

> "这个月我们不和别人比，和自己比。设三个档：达标奖、突破奖、王者奖。达标是本分，突破是本事，王者是传说。我相信咱们团队至少出两个王者，是谁我先不猜，月底见分晓！"

### 新人鼓励

> "新人第一个月不用看业绩，看这三点：①话术语气是否越来越自然 ②对客户的判断是否越来越准 ③是否敢主动邀约了。这三点做到了，业绩是迟早的事。"
