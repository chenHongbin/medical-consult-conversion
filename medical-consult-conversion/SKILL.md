# 医疗咨询转化技能

基于赖静茹老师20年大健康行业咨询培训实战经验，覆盖6大模块。

## 核心转化公式

> **业绩 = 有效对话量 × 预约转化率 × 到院转化率 × 心态能量系数**
>
> **业绩 = 流量 × 转化 × 复盘迭代**

两个公式互补：前者聚焦个人执行力，后者聚焦团队运营力。

### 流量端关键认知

想要在原流量基础上提转化，必须基于**流量平台特性**做转化策略：
- 分析平台、分析用户特性，为转化做指导（可用AI多维分析）
- 各平台总有高转化数据，把高转化对话拉出来匹配、拆解、找规律
- **南抖北快**——不同地域用户习惯不同，转化策略也应本土化

### 高效咨询4个优秀特质

1. **聚焦** — 聊天时全神贯注，一心不二用
2. **调频** — 能掌控自己的思维、情绪与行动
3. **执行** — 把"我觉得"变成"我来做"
4. **迭代** — 定期复盘，持续优化话术和流程

### 能量管理优先级

> 能力很重要，但**能量比能力更重要**。
> 低能量状态下，客户信任度直接下降30%。

## 用户画像与场景判断

**首次沟通用户**：
- 先破冰建立信任 → 用话术库破冰类话术
- 再挖掘需求 → 用需求探询类话术
- 最后引导预约 → 用邀约转化类话术

**已沟通未预约用户**：
- 直接走回访SOP → 参考 `followup.md`

**抗拒/犹豫用户**：
- 先用安抚话术化解抗拒 → 参考 `scripts.md`
- 再用价值塑造话术 → 参考 `scripts.md`
- 心态层面辅助 → 参考 `mindset.md`

**团队管理者**：
- 新人培训 → 参考 `training.md`
- 日常激励 → 参考 `mindset.md` + `scripts.md`
- AI提效 → 参考 `ai-empower.md`

## 团队管理四象限

带团队高转化，先识别成员状态，再针对性施策：

| 意愿 \ 能力 | 高能力 | 低能力 |
|---|---|---|
| **高意愿** | 授权+肯定，团队扛把子，放手去干 | 培养多维能力，拆解对话，给话术技巧升级 |
| **低意愿** | 状态/能量/心力问题，提前干预，一对一沟通 | 拉不起来则各自安好，能拉则尽力拉 |

> 管理的本质是让人们的优势变得富有成效，让弱点变得无关紧要。——彼得·德鲁克

## 核心方法论索引

- **咨询约到8步法 + 多维约到法**：赖老师20年经验总结的底层转化逻辑，用于对话拆解、卡点分析
- **三段式安抚框架**：共情句 → 缓冲句 → 专业转入（详见 `scripts.md`）
- **培训落地三步法**：提炼3个黄金关键点 → 刻意练习3-7天 → 每日5分钟复盘（详见 `training.md`）
- **AI话术生成三步法**：给AI设定角色 → 描述场景生成话术 → 筛选优化落地（详见 `ai-empower.md`）

## 输出原则

1. **话输出即落地** — 所有话术可直接复制使用，不做空泛理论
2. **场景化** — 结合用户具体业务场景（口腔/医美/植发等）调整话术
3. **模块化组合** — 根据用户实际需求组合多个模块内容
4. **保持赖老师风格** — 接地气、有能量、直接实用
